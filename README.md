## Descripción

Este proyecto esta creado en Nestjs, actualmente el microservicio de reservas para NurBNB se enfoca en realizar las reservas de propiedades previamente registradas

## Instalación

```bash
$ npm install
```

## Correr la aplicación

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# integration tests
$ npm run test:integration
```
